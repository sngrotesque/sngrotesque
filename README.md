你好，我是 **SN-Grotesque**，也可以叫我 **栀子鱼鱼花**。  
欢迎来到我的 GitHub。

我是一名开发者、音乐人、视频创作者，同时也从事游戏开发。  
我使用 **Unity** 与 **Unreal Engine** 制作游戏，用 **FL Studio** 创作音乐，日常编码主要使用 **Visual Studio 2022** 与 **VSCode**。

在常规快速开发中，我偏好 **MSYS2 - MinGW64 / GCC** 环境。  
Python 方面，我目前固定使用 **3.11.9**。

我喜欢研究底层技术、算法实现、密码学与网络安全，追求用最短的代码实现最高的质量。
