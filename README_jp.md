こんにちは、**SN-Grotesque** です。**栀子鱼鱼花**とも呼んでください。  
私の GitHub へようこそ。

私は開発者、ミュージシャン、動画クリエイターであり、ゲーム開発も行っています。  
ゲーム制作には **Unity** と **Unreal Engine**、音楽制作には **FL Studio**、日常的なコーディングには **Visual Studio 2022** と **VSCode** を使用しています。

高速な開発が必要な場面では、**MSYS2 - MinGW64 / GCC** 環境を好んで使います。  
Python は現在 **3.11.9** を固定で使用しています。

低レベルの技術、アルゴリズム実装、暗号技術、ネットワークセキュリティを研究するのが好きで、最小のコードで最高の品質を追求しています。

# プロジェクト

1. [The Wizard's Universal Key](https://github.com/sngrotesque/wizard_key)  
   すべてをやろうとするのは無謀で、当時の自分はあまりにも幼稚でした。このリポジトリ（プロジェクト）は少なくとも3年間書き続けましたが、結局放棄しました。  
   もう以前のように時間を費やす価値がないと気づき、専門分野に集中すべきだと悟りました。  
   いつかまた再開するかもしれませんが、それはずっと先のことでしょう。  
   しかし、プロジェクト管理やコーディングの面で本当に多くの成長をもたらしてくれました。

2. [Cheat kernel](https://github.com/sngrotesque/CheatKernel)  
   現在作成中の汎用ゲームチート解析ツールです。現時点ではオープンソース化の予定はありません。ご期待ください。  
   **Unreal Engine / Unity** で開発されたゲームを対象としています。  
   可能であれば、できるだけ汎用的なチートツールにしたいですが、それは難しいですね。

3. [Common Vulnerability Exploit](https://github.com/sngrotesque/CVE_CommonVulnerabilityExploit)  
   ここには私が収集したすべての Nday 脆弱性の PoC と Exploit が集められています。ほとんどは CVE (Common Vulnerabilities and Exposures) から取得したもので、一部は CVE 番号を持っていません。
