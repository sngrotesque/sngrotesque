안녕하세요, 저는 **SN-Grotesque**입니다. **치쯔위위화**라고도 부를 수 있습니다.  
제 GitHub에 오신 것을 환영합니다.

저는 개발자, 음악가, 영상 제작자이며 게임 개발도 하고 있습니다.  
게임 제작에는 **Unity**와 **Unreal Engine**, 음악 창작에는 **FL Studio**, 일상 코딩에는 **Visual Studio 2022**와 **VSCode**를 사용합니다.

빠른 개발이 필요할 때는 **MSYS2 - MinGW64 / GCC** 환경을 선호합니다.  
Python은 현재 **3.11.9** 버전을 고정적으로 사용하고 있습니다.

저는 로우레벨 기술, 알고리즘 구현, 암호학 및 네트워크 보안을 연구하는 것을 좋아하며, 가장 짧은 코드로 최고의 품질을 추구합니다.

# 프로젝트

1. [The Wizard's Universal Key](https://github.com/sngrotesque/wizard_key)  
   모든 것을 하려는 것은 불가능하다는 것을 깨달았습니다. 당시의 저는 너무 순진했습니다. 이 저장소(프로젝트)는 적어도 3년 동안 작성했지만 결국 포기했습니다.  
   더 이상 예전처럼 시간을 투자할 가치가 없다는 것을 알게 되었고, 세부 분야에 집중해야 한다고 생각했습니다.  
   언젠가 다시 시작할 수도 있겠지만, 그것은 아주 먼 미래의 일일 것입니다.  
   하지만 이 프로젝트는 프로젝트 관리와 코드 작성 측면에서 정말 많은 성장을 안겨주었습니다.

2. [Cheat kernel](https://github.com/sngrotesque/CheatKernel)  
   현재 작성 중인 범용 게임 치트 분석 도구입니다. 현재로서는 오픈소스화를 고려하지 않고 있으니 기대해 주십시오.  
   **Unreal Engine / Unity**로 개발된 게임에 초점을 맞출 것입니다.  
   가능하다면 최대한 범용적인 게임 치트 도구로 만들고 싶지만, 그것은 어려운 일이지 않습니까?

3. [Common Vulnerability Exploit](https://github.com/sngrotesque/CVE_CommonVulnerabilityExploit)  
   여기에는 제가 수집한 모든 Nday 취약점 PoC와 Exploit이 모여 있습니다. 대부분은 CVE(Common Vulnerabilities and Exposures) 공개 자료에서 가져왔으며, 일부는 CVE 번호가 없습니다.
